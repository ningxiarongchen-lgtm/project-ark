# 📋 SF系列数据字段快速参考

**最后更新**: 2025-11-06

---

## 🚀 快速开始

### SF系列模板包含 25 个字段

```
1. model_base          型号（如：SF10-150DA）
2. series              系列（SF）
3. mechanism           机构类型（Scotch Yoke - 拨叉式）
4. valve_type          阀门类型（Ball Valve/Butterfly Valve）
5. body_size           本体尺寸（SF10, SF12等）
6. cylinder_size       气缸尺寸（150, 170, 200等）
7. action_type         作用类型（DA双作用/SR单作用）
8. spring_range        弹簧范围（SR3/SR4/SR5/SR6）
9. base_price          基础价格（向后兼容）
10. base_price_normal  常温价格（-20°C ~ 80°C）
11. base_price_low     低温价格（-60°C ~ -20°C）
12. base_price_high    高温价格（80°C ~ 200°C）
13. torque_symmetric   对称拨叉扭矩（JSON格式）
14. torque_canted      偏心拨叉扭矩（JSON格式）
15. connect_flange     连接法兰（ISO 5211 F10等）
16. L1                 总长度（mm）
17. L2                 气缸长度（mm）
18. m1                 机身宽度1（mm）
19. m2                 机身宽度2（mm）
20. A                  安装尺寸A（mm）
21. H1                 高度H1（mm）
22. H2                 高度H2（mm）
23. D                  直径D（mm）
24. G                  气动接口（NPT1/4"等）
25. description        描述（可选）
```

## 📊 双作用(DA)vs单作用(SR)

### DA - Double Acting（双作用）

**特点**: 
- 气动开/气动关
- 需要持续气源
- 扭矩更大，响应更快

**型号示例**: SF10-150**DA**, SF12-170**DA**

**扭矩数据格式**:
```json
{
  "0.3_0": 309,    // 0.3MPa，0°角
  "0.3_45": 185,   // 0.3MPa，45°角
  "0.3_90": 309,   // 0.3MPa，90°角
  "0.4_0": 412,
  "0.4_45": 247,
  "0.4_90": 412,
  "0.5_0": 515,
  "0.5_45": 309,
  "0.5_90": 515,
  "0.6_0": 618,
  "0.6_45": 371,
  "0.6_90": 618
}
```

### SR - Spring Return（弹簧复位/单作用）

**特点**:
- 气动开/弹簧关（或反向）
- 气源失效时自动复位
- 安全性高

**型号示例**: SF10-150**SR3**, SF12-170**SR4**

**弹簧范围**:
- **SR3**: 3Bar气源压力
- **SR4**: 4Bar气源压力
- **SR5**: 5Bar气源压力
- **SR6**: 6Bar气源压力

**扭矩数据格式**:
```json
{
  "sst": 187,       // Spring Start Torque
  "srt": 91,        // Spring Run Torque
  "set": 118,       // Spring End Torque
  "ast_0.3": 183,   // Air Start @ 0.3MPa
  "art_0.3": 89,    // Air Run @ 0.3MPa
  "aet_0.3": 115,   // Air End @ 0.3MPa
  "ast_0.4": 293,
  "art_0.4": 155,
  "aet_0.4": 225,
  "ast_0.5": 396,
  "art_0.5": 217,
  "aet_0.5": 328
}
```

## 🎯 对称拨叉 vs 偏心拨叉

### 对称拨叉（Symmetrical Yoke）- torque_symmetric

**适用阀门**: 
- ✅ **Ball Valve**（球阀）
- ✅ 对称式蝶阀

**特点**: 
- 0°和90°位置扭矩相同
- 45°位置扭矩最小
- 适合需要均衡扭矩的场合

**型号标识**: 不带"C"后缀（如SF10-150DA）

### 偏心拨叉（Canted Yoke）- torque_canted

**适用阀门**: 
- ✅ **Butterfly Valve**（蝶阀）
- ✅ 偏心式阀门

**特点**: 
- 0°位置扭矩最大
- 90°位置扭矩中等
- 45°位置扭矩最小
- 适合蝶阀的开关特性

**型号标识**: 带"C"后缀（如SF10/C-150DA）

## 💰 价格字段说明

### 价格体系

| 字段 | 温度范围 | 计算方式 | 示例 |
|------|---------|---------|------|
| `base_price_normal` | -20°C ~ 80°C | 标准价格 | 1339元 |
| `base_price_low` | -60°C ~ -20°C | 常温价×1.05 | 1405元 |
| `base_price_high` | 80°C ~ 200°C | 常温价×1.05 | 1405元 |

### Excel中如何填写

**方案1**: 只填常温价格，系统自动计算
```
base_price_normal: 1339
base_price_low: （空）  ← 自动计算为1405
base_price_high: （空） ← 自动计算为1405
```

**方案2**: 全部填写（推荐）
```
base_price_normal: 1339
base_price_low: 1405
base_price_high: 1405
```

## 📏 尺寸字段说明

### 关键尺寸

```
        ← L1 (总长度) →
        ┌─────────────────┐
        │  ←  L2 →       │
    ↑   │  ┌─────┐       │
    │   │  │     │       │
   H1   │  │ ⚙  │       │  
    │   │  │     │       │
    ↓   │  └─────┘       │
        └─────────────────┘
          ← m1/m2 →
```

| 字段 | 说明 | DA示例 | SR示例 |
|------|------|--------|--------|
| `L1` | 总长度 | 350mm | 350mm |
| `L2` | 气缸长度 | 127mm | 467mm |
| `H1` | 安装高度 | 82mm | 82mm |
| `D` | 外径 | 207mm | 207mm |
| `m1` | 机身宽度1 | 76mm | 76mm |
| `m2` | 机身宽度2 | 143.5mm | 143.5mm |

**注意**: SR型（单作用）的L2通常比DA型长，因为需要容纳弹簧

```
SF12系列: ISO 5211 F12
SF14系列: ISO 5211 F14
SF16系列: ISO 5211 F16
SF25系列: ISO 5211 F25
SF30系列: ISO 5211 F30
SF35系列: ISO 5211 F35
SF40系列: ISO 5211 F40
SF48系列: ISO 5211 F48
SF60系列: ISO 5211 F60
```

### 气动接口 (G)

**NPT螺纹标准**:
```
SF10-SF16系列: NPT1/4" 或 NPT3/8"
SF25-SF35系列: NPT3/4"
SF40-SF60系列: NPT1" 或 NPT2"
```

## ⚠️ 常见错误

### 错误1: 扭矩JSON格式不正确

❌ **错误**:
```
{0.3_0:309}           // 键没有引号
{'0.3_0':309}         // 单引号
{ "0.3_0" : 309 }     // 多余空格
```

✅ **正确**:
```
{"0.3_0":309,"0.3_45":185,"0.3_90":309}
```

### 错误2: 气缸尺寸类型错误

❌ **错误**:
```
cylinder_size: "150"   // 字符串
```

✅ **正确**:
```
cylinder_size: 150     // 数字
```

### 错误3: 价格字段混用

⚠️ **不推荐**:
```
base_price: 1339           // 老字段
base_price_normal: （空）
```

✅ **推荐**:
```
base_price_normal: 1339    // 新字段
base_price_low: 1405
base_price_high: 1405
```

## 🎯 使用流程

### 1. 下载模板

```
管理员登录 → 数据管理 → 执行器管理 → 下载SF模板
```

### 2. 填写数据

- 参考官方技术手册填写扭矩数据
- 参考产品样册填写尺寸数据
- 参考价格表填写价格数据

### 3. 上传导入

```
选择填好的Excel文件 → 点击导入 → 系统验证 → 导入成功
```

### 4. 技术选型验证

```
技术工程师登录 → 新建项目 → 智慧选型 → 选择执行器
→ 系统自动匹配SF系列产品 → 显示价格和尺寸
```
