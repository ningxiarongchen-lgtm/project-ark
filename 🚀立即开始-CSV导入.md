## ⚡ 3步导入（不到2分钟）

### 步骤1：打开导入页面
```
登录系统 → 产品批量导入
```

### 步骤2：选择执行器导入
1. 选择：**执行器数据（AT/GY/SF系列）**
2. 点击：**选择文件（CSV/Excel）**
3. 选择：`at_gy_actuators_data_final.csv` 或 `sf_actuators_data.csv`
4. **建议**：勾选"更新重复数据"（如果想覆盖已有数据）

### 步骤3：开始导入
1. 点击：**开始导入**
2. 等待30-60秒
3. ✅ 查看结果！
