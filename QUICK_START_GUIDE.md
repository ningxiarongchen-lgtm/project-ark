## 💡 核心功能使用

### 1️⃣ 单个智能选型

**操作步骤**：
```
1. 在工作台找到目标项目
2. 点击"智能选型"按钮
3. 填写选型参数：
   ├─ 阀门类型（球阀/蝶阀）
   ├─ 阀门尺寸（DN50-DN400）
   ├─ 扭矩值（Nm）
   ├─ 作用类型（DA/SR）
   └─ 故障位置（仅SR需要）
4. 点击"开始选型"
5. 查看推荐结果
```

**注意事项**：
- ✅ 工作角度自动设为90度
- ✅ 安全系数默认1.3倍
- ✅ 球阀自动使用对称轭架
- ✅ 蝶阀自动使用倾斜轭架

### 2️⃣ 批量智能选型（推荐）


| 字段 | 说明 | 示例 |

| 阀门类型 | 球阀/蝶阀 | 球阀 |
| 阀门尺寸 | 公称直径 | DN100 |
| 扭矩(Nm) | 阀门扭矩 | 500 |
| 作用类型 | DA/SR | DA |

**SR额外字段**：
| 字段 | 说明 | 示例 |
|------|------|------|
| 故障安全位置 | Fail Close/Open | Fail Close |
| 开启扭矩(Nm) | 开阀扭矩 | 400 |
| 关闭扭矩(Nm) | 关阀扭矩 | 450 |

**可选字段**：
| 字段 | 说明 | 默认值 |
|------|------|--------|
| 安全系数 | 安全裕度 | 1.3 |
| 机构类型 | SF/AT/GY | Scotch Yoke |
| 数量 | 订购数量 | 1 |
| 工艺介质 | 应用描述 | - |

#### 步骤3：上传选型

```
1. 返回批量选型页面
2. 点击"上传Excel文件"

6. 点击"开始批量选型"
7. 等待处理完成

```

## 📋 选型类型详解

### DA（双作用）执行器

**定义**: 气源驱动两个方向，失气停在当前位置

**何时使用**：
- ✅ 无需故障安全功能
- ✅ 成本敏感项目
- ✅ 一般应用场合

**Excel填写示例**：
```
作用类型: DA
故障安全位置: （留空）
开启扭矩: （留空）
关闭扭矩: （留空）
```

---

### SR（单作用）执行器

**定义**: 弹簧提供单向力，气源提供反向力

**何时使用**：
- ✅ 需要故障安全保护
- ✅ 危险介质
- ✅ 关键工艺位置

#### 🔴 故障关 (Fail Close - STC)

**定义**: 失气时弹簧关闭阀门

**应用场景**：
- 有毒气体管道
- 易燃易爆介质
- 需要紧急切断的场合

**Excel填写示例**：
```
作用类型: SR
故障安全位置: Fail Close
开启扭矩: 400
关闭扭矩: 450
```

**推荐型号格式**: `SF-SR52K8-STC` 或 `SF-SR52K8/C-STC`

#### 🟢 故障开 (Fail Open - STO)

**定义**: 失气时弹簧打开阀门

**应用场景**：
- 冷却水管道
- 消防水系统
- 需要紧急开启的场合

**Excel填写示例**：
```
作用类型: SR
故障安全位置: Fail Open
开启扭矩: 350
关闭扭矩: 380
```

**推荐型号格式**: `SF-SR52K8-STO` 或 `SF-SR52K8/C-STO`

## 🔧 执行器系列选择

### SF系列（Scotch Yoke 拨叉式）

**特点**：
- 🎯 高扭矩输出
- 💪 结构坚固
- ⚙️ 维护简单

**适用**：
- 大扭矩场合
- 恶劣环境
- 长期运行

**轭架类型**：
```
球阀 → 对称轭架 → SF-DAxx
蝶阀 → 倾斜轭架 → SF-DAxx/C
```

---

### AT/GY系列（Rack & Pinion 齿轮齿条式）

**特点**：
- ⚡ 响应速度快
- 📏 体积紧凑
- 🎛️ 控制精度高

**材质选择**：
```
AT系列: 铝合金 → 轻量级应用
GY系列: 不锈钢 → 腐蚀性环境
```

**适用**：
- 快速响应场合
- 空间受限位置
- 精确控制需求

## ❓ 常见问题

### Q1: 如何选择DA还是SR？

**A**: 判断标准
```
需要故障安全？
├─ 是 → SR（单作用）
│  └─ 失气时需要关闭？
│     ├─ 是 → Fail Close (STC)
│     └─ 否 → Fail Open (STO)
└─ 否 → DA（双作用）
```

### Q2: SR只提供了一个扭矩值怎么办？

**A**: 系统会自动处理
- 如果只填"扭矩"字段，开启和关闭都用这个值
- 如果只填"开启扭矩"，关闭扭矩=开启扭矩
- 如果只填"关闭扭矩"，开启扭矩=关闭扭矩

### Q3: 球阀和蝶阀有什么区别？

**A**: 自动轭架匹配
```
球阀：
├─ 扭矩曲线对称
├─ 使用对称轭架
└─ 型号不带 /C

蝶阀：
├─ 起点终点扭矩大
├─ 使用倾斜轭架
└─ 型号带 /C
```

### Q4: 安全系数如何设置？

**A**: 推荐值
```
常规环境: 1.3倍（默认）
高安全要求: 1.5倍
特殊工况: 1.2-2.0倍
```

### Q5: 工作角度需要填吗？

**A**: 不需要
- 系统自动设置为90度
- 适用于所有球阀和蝶阀

## 🎯 最佳实践

### ✅ DO（推荐做法）

1. **使用批量选型**
   - 提高效率
   - 减少错误
   - 便于管理

2. **准确填写扭矩**
   - 参考阀门制造商数据
   - 考虑介质特性
   - 预留安全裕度

3. **合理选择SR**
   - 评估风险等级
   - 确认故障安全位置
   - 分别提供开/关扭矩

4. **检查选型结果**
   - 验证推荐等级
   - 确认扭矩裕度
   - 核对型号命名

### ❌ DON'T（避免做法）

1. **不要随意修改模板结构**
   - 保持表头完整
   - 不要删除列
   - 按照说明填写

2. **不要忽略安全系数**
   - 使用默认1.3倍
   - 高风险场合增加
   - 不要设置过小

3. **不要混淆DA和SR**
   - 明确故障安全需求
   - 正确填写作用类型
   - SR必须填故障位置

4. **不要忽视材质选择**
   - 考虑介质腐蚀性
   - 评估温度范围
   - 选择合适系列
