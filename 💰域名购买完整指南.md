## 📖 阿里云购买流程（推荐，最详细）

### 第1步：注册阿里云账号（3分钟）

1. **访问阿里云**
   ```
   https://account.aliyun.com/register/
   ```

2. **填写注册信息**
   - 手机号
   - 验证码
   - 设置密码
   - 点击"同意协议并注册"

3. **验证手机号**
   - 输入短信验证码

✅ 完成！

---

### 第2步：搜索并选择域名（5分钟）

1. **访问域名注册页面**
   ```
   https://wanwang.aliyun.com/domain
   ```

2. **搜索你想要的域名**
   ```
   例如输入：mycompany
   ```
   
3. **查看搜索结果**
   - 绿色✅ = 可以注册
   - 红色❌ = 已被注册
   
4. **选择域名后缀**
   - 推荐：`.com` 或 `.cn`（专业）
   - 便宜：`.top` 或 `.xyz`（实惠）

5. **点击"加入清单"**

**💡 域名命名建议：**
- 短小精悍（6-15个字符）
- 容易记忆
- 与公司/项目相关
- 避免数字和连字符（-）
- 例如：
  - `smartmfg.com`（智能制造）
  - `iot-system.cn`（物联网系统）
  - `projectark.top`（方舟项目）

---

### 第3步：下单购买（2分钟）

1. **点击右上角"购物车"**

2. **选择购买年限**
   - 1年 - 适合测试
   - 3年 - 推荐（通常有折扣）
   - 5年 - 长期项目

3. **勾选同意条款**

4. **点击"立即购买"**

5. **选择支付方式**
   - 支付宝（推荐）
   - 微信支付
   - 银行卡

6. **完成支付**

✅ 购买完成！

---

### 第4步：实名认证（必须，1-3天）

**⚠️ 重要：域名必须实名认证才能使用！**

1. **登录阿里云控制台**
   ```
   https://dc.console.aliyun.com
   ```

2. **找到你的域名**
   - 状态显示"未实名认证"

3. **点击"实名认证"**

4. **选择认证类型**
   
   **个人认证（推荐，快）：**
   - 姓名
   - 身份证号
   - 上传身份证照片（正反面）
   - 审核时间：1-3小时
   
   **企业认证（正式）：**
   - 企业名称
   - 统一社会信用代码
   - 营业执照照片
   - 审核时间：1-3天

5. **提交审核**

6. **等待审核通过**
   - 阿里云会发短信通知
   - 通常1-3小时内通过

✅ 实名认证完成！

---

### 第5步：配置Cloudflare（10分钟）

**等待实名认证通过后，立即配置！**

#### 5.1 注册Cloudflare

1. **访问Cloudflare**
   ```
   https://dash.cloudflare.com/sign-up
   ```

2. **注册账号**
   - 邮箱
   - 密码
   - 验证邮箱

#### 5.2 添加域名到Cloudflare

1. **点击"Add a Site"**

2. **输入你刚购买的域名**
   ```
   例如：mycompany.com
   ```

3. **选择"Free"计划**

4. **点击"Continue"**

5. **等待DNS扫描**（30秒）

#### 5.3 修改域名服务器（关键！）

1. **Cloudflare会显示两个域名服务器**
   ```
   例如：
   amber.ns.cloudflare.com
   reza.ns.cloudflare.com
   ```
   **📝 记下这两个地址！**

2. **回到阿里云控制台**
   ```
   https://dc.console.aliyun.com
   ```

3. **找到你的域名，点击"管理"**

4. **点击左侧"DNS修改"**

5. **点击"修改DNS服务器"**

6. **输入Cloudflare的两个域名服务器**
   - DNS服务器1：`amber.ns.cloudflare.com`
   - DNS服务器2：`reza.ns.cloudflare.com`

7. **点击"确定"**

8. **等待生效**（5分钟-24小时，通常30分钟）

#### 5.4 在Cloudflare配置DNS记录

1. **回到Cloudflare Dashboard**

2. **点击你的域名**

3. **点击"DNS"选项卡**

4. **添加DNS记录**

   **记录1：主域名**
   ```
   Type: CNAME
   Name: @
   Target: project-ark-one.vercel.app
   Proxy status: Proxied（橙色云朵🟠）
   TTL: Auto
   ```
   点击"Save"

   **记录2：www子域名**
   ```
   Type: CNAME
   Name: www
   Target: project-ark-one.vercel.app
   Proxy status: Proxied（橙色云朵🟠）
   TTL: Auto
   ```
   点击"Save"

✅ Cloudflare配置完成！

---

### 第6步：Vercel添加自定义域名（3分钟）

1. **登录Vercel**
   ```
   https://vercel.com/dashboard
   ```

2. **进入项目**
   - 找到 `project-ark-one`
   - 点击项目名

3. **进入设置**
   - 点击顶部"Settings"
   - 点击左侧"Domains"

4. **添加域名**
   
   **添加主域名：**
   - 在输入框输入：`mycompany.com`
   - 点击"Add"
   
   **添加www域名：**
   - 在输入框输入：`www.mycompany.com`
   - 点击"Add"

5. **等待验证**
   - 绿色✅ = 配置成功
   - 黄色⚠️ = 正在验证（等待几分钟）
   - 红色❌ = 配置错误（检查DNS设置）

✅ Vercel配置完成！

---

### 第7步：优化Cloudflare设置（可选，5分钟）

#### 7.1 SSL/TLS设置

1. **在Cloudflare，点击"SSL/TLS"**

2. **设置加密模式**
   - 选择：**Full (strict)**

3. **开启"Always Use HTTPS"**
   - 开关设为"On"

4. **开启"Automatic HTTPS Rewrites"**
   - 开关设为"On"

#### 7.2 性能优化

1. **点击"Speed" → "Optimization"**

2. **开启自动压缩**
   - ✅ Auto Minify: JavaScript
   - ✅ Auto Minify: CSS
   - ✅ Auto Minify: HTML
   - ✅ Brotli

3. **点击"Caching"**
   - Caching Level: Standard
   - Browser Cache TTL: 4 hours

✅ 优化完成！

---

### 第8步：测试访问（5-30分钟后）

1. **等待DNS传播**（5-30分钟）

2. **访问你的域名**
   ```
   https://mycompany.com
   ```

3. **在电脑测试**
   - Chrome浏览器
   - 应该能看到系统登录页面

4. **在手机测试**（重要！）
   - 使用移动网络（4G/5G）
   - 打开Safari/Chrome
   - 访问你的域名
   - 应该速度很快！✅

✅ 部署成功！🎉

## ❓ 常见问题

### Q1: 必须实名认证吗？
**A:** 是的，中国注册的域名必须实名认证才能使用。审核时间1-3小时。

### Q2: .cn 域名需要备案吗？
**A:** 如果服务器在中国，需要备案。但我们用Vercel（海外）+ Cloudflare，不需要备案。

### Q3: 域名多久生效？
**A:** 
- 购买后：立即拥有
- 实名认证：1-3小时
- DNS修改：5分钟-24小时（通常30分钟）
- 完整生效：1-2小时

### Q4: 可以转移域名吗？
**A:** 可以。购买60天后可以转移到其他注册商。

### Q5: 忘记续费会怎样？
**A:** 
- 到期前30天：阿里云会提醒
- 到期后30天：域名暂停解析，但可以续费恢复
- 到期后60天：域名进入赎回期，需要高价赎回
- 到期后75天：域名释放，任何人可以注册
