# ⚡ 阿里云OSS - 5分钟快速部署

## 🎯 超简化版（先跑起来再说）

### 第1步：注册阿里云（1分钟）
```
访问: https://www.aliyun.com
点击"免费注册" → 手机号注册 → 短信验证
```

### 第2步：开通OSS（1分钟）
```
访问: https://oss.console.aliyun.com
点击"立即开通" → 按量付费 → 确定
```

### 第3步：创建Bucket（1分钟）
```
点击"创建Bucket"
名称: smart-system-2024 (或其他唯一名称)
地域: 华东1-杭州
读写权限: 公共读 ← 重要！
点击"确定"
```

### 第4步：构建前端（2分钟）
```bash
# 在终端执行
cd "/Users/hexiaoxiao/Desktop/Model Selection System/frontend"
npm run build
```

### 第5步：上传文件（最简单方式）
```
1. OSS控制台 → 点击你的Bucket → 文件管理
2. 点击"上传文件"
3. 进入 frontend/dist 目录
4. 全选所有文件和文件夹
5. 拖拽上传
```

### 第6步：配置静态网站（30秒）
```
Bucket管理 → 基础设置 → 静态页面 → 设置
默认首页: index.html
默认404页: index.html
保存
```

### 第7步：获取地址并访问
```
Bucket概览 → 找到"外网访问地址"
复制地址 → 手机打开测试
```
