# ⚡ 快速部署指南

## 🚀 一键部署（最简单）

改完代码后，在项目根目录运行：

```bash
./deploy.sh "你的修改说明"
```

### 📋 示例

```bash
# 修复bug
./deploy.sh "fix: 修复Dashboard统计问题"

# 添加新功能
./deploy.sh "feat: 添加用户管理功能"

# 优化性能
./deploy.sh "perf: 优化页面加载速度"

# 更新文档
./deploy.sh "docs: 更新部署文档"
```

---

## ✨ 这个命令做了什么？

1. ✅ **Git add** - 添加所有更改的文件
2. ✅ **Git commit** - 提交代码到本地仓库
3. ✅ **Git push** - 推送到GitHub远程仓库
4. ✅ **npm run build** - 构建前端项目
5. ✅ **部署** - 自动部署到Cloudflare Pages
6. ✅ **显示地址** - 显示部署后的访问地址

---

## 🌐 部署后访问

**主地址：** https://smart-system.pages.dev

**预览地址：** 每次部署会生成新的预览URL

**提示：** 部署完成后等待2-3分钟，然后强制刷新浏览器：
- **Mac**: `Cmd + Shift + R`
- **Windows**: `Ctrl + Shift + R`

## ❓ 常见问题

### 1. 脚本没有执行权限？
```bash
chmod +x deploy.sh
```

### 2. 部署失败？
- 检查网络连接
- 确保已登录 `wrangler login`
- 查看错误信息进行排查

### 3. 需要只部署不提交Git？
```bash
./deploy-cloudflare.sh
```
