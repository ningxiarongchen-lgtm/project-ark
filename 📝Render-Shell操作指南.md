# 📝 Render Shell 操作指南

## 🎯 目标
在Render云端创建测试账号（密码会正确hash）

---

## 📋 操作步骤

### 第1步：打开Render Shell

1. **打开Render网页**：https://dashboard.render.com
2. **点击左侧菜单的 "Shell"** ⚡（在"Environment"下面）
3. 等待Shell加载完成

### 第2步：运行初始化脚本

在Shell命令行中输入：

```bash
node seed_final_acceptance.js
```

**按回车执行**

## ⚠️ 注意事项

- **这会替换现有用户数据**
- 但会保留：执行器、配件、供应商、项目数据
- 只是重新创建用户账号，确保密码正确

---

## 🔄 为什么要这样做？

迁移的用户密码hash格式可能不匹配。
运行seed脚本会：
1. 创建10个标准测试账号
2. 密码正确hash
3. 确保能登录
4. 保留其他所有数据（337个执行器、10个配件等）
