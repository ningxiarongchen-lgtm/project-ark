# ⚡ 快速上传数据到生产环境

## 🎯 3 步完成数据上传

---

### 第 1 步：获取生产环境 MongoDB URI

#### 方法 A：从 Render 获取

```
1. 访问: https://dashboard.render.com
2. 选择后端服务
3. 点击 "Environment" 标签
4. 复制 MONGODB_URI 的值
```

#### 方法 B：从 MongoDB Atlas 获取

```
1. 访问: https://cloud.mongodb.com
2. 点击 "Connect" → "Connect your application"
3. 复制连接字符串
```

---

### 第 2 步：运行上传脚本

打开终端，执行：

```bash
cd "/Users/hexiaoxiao/Desktop/Model Selection System"

```
