## 🆘 遇到问题？

### 无法启动
```bash
# 检查MongoDB是否运行
brew services start mongodb-community

# 重新启动
bash 演示环境启动.sh
```

### 数据混乱
```bash
# 重新初始化数据
cd backend
npm run seed:final
```

### 页面显示异常
按 `Cmd+Shift+R` (Mac) 强制刷新浏览器
