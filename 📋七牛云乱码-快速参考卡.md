浏览器不知道用什么编码解析
手机浏览器猜错了编码

步骤1: https://portal.qiniu.com 登录

步骤2: 对象存储 → 你的空间 → 空间设置

步骤3: 自定义响应头配置 → 添加响应头

步骤4: 添加3个设置：
  ┌─────────────────────────────────────┐
  │ 名称: Content-Type                   │
  │ 值: text/html; charset=utf-8         │
  │ 文件: *.html                         │
  └─────────────────────────────────────┘
  ┌─────────────────────────────────────┐
  │ 名称: Content-Type                   │
  │ 值: application/javascript; charset=utf-8│
  │ 文件: *.js                           │
  └─────────────────────────────────────┘
  ┌─────────────────────────────────────┐
  │ 名称: Content-Type                   │
  │ 值: text/css; charset=utf-8          │
  │ 文件: *.css                          │
  └─────────────────────────────────────┘

步骤5: CDN → 刷新预热 → 刷新你的URL

步骤6: 清除手机缓存 → 等2分钟 → 重新访问

✅ 完成！
