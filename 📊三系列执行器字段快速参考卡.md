## 📊 SF系列（Scotch Yoke 拨叉式）

### 字段总数：24个

### 核心字段
```yaml
型号识别:
  - model_base          # 基础型号 (如: SF10-150DA)
  - series              # 系列 (SF)
  - mechanism           # 机构 (Scotch Yoke)
  - valve_type          # 阀门类型 (Ball Valve)

规格配置:
  - body_size           # 本体尺寸 (如: SF10)
  - cylinder_size       # 气缸尺寸 (如: 150)
  - action_type         # 作用方式 (DA/SR)
  - spring_range        # 弹簧范围 (如: SR3)

价格体系:
  - base_price          # 基础价格
  - base_price_normal   # 常温价格
  - base_price_low      # 低温价格
  - base_price_high     # 高温价格

扭矩数据 (JSON格式):
  - torque_symmetric    # 对称扭矩 (不同气压下的扭矩值)
  - torque_canted       # 偏置扭矩 (不同气压下的扭矩值)
  
连接尺寸:
  - connect_flange      # 连接法兰 (ISO 5211标准)
  - L1, L2              # 长度尺寸
  - m1, m2              # 安装尺寸
  - A, H1, H2, D, G     # 其他尺寸参数
```

### 🎯 SF系列独有字段
- `cylinder_size` - 气缸尺寸（150, 200等）
- `base_price` - 独立基础价格字段
- `torque_symmetric` - 对称扭矩JSON数据
- `torque_canted` - 偏置扭矩JSON数据
- `connect_flange` - ISO 5211连接法兰
- `L1, L2, m1, m2, A, H1, H2, D, G` - 详细连接尺寸

### 💡 SF特殊说明
- **扭矩数据格式**：JSON对象，键格式为 `气压_角度` (如: "0.3_0", "0.4_45")
- **价格策略**：支持常温/低温/高温三档价格，可选填写
- **作用类型**：DA（双作用）、SR（单作用）
- **弹簧范围**：仅单作用型号需要（如: SR3, SR4）

## 📊 AT系列（Rack & Pinion 齿轮齿条式）

### 字段总数：20个

### 核心字段
```yaml
型号识别:
  - model_base          # 基础型号 (如: AT-SR52K8)
  - series              # 系列 (AT)
  - mechanism           # 机构 (Rack & Pinion)
  - valve_type          # 阀门类型 (Ball Valve)

规格配置:
  - action_type         # 作用方式 (SR/DA)
  - spring_range        # 弹簧范围 (K8, K10等)
  - body_size           # 本体尺寸 (AT-052, AT-063等)

价格体系:
  - base_price_normal   # 常温价格
  - base_price_low      # 低温价格
  - base_price_high     # 高温价格

附件配置:
  - manual_override_model   # 手操器型号
  - manual_override_price   # 手操器价格
  - spare_parts_model       # 备件型号
  - spare_parts_price       # 备件价格

法兰连接:
  - flange_standard     # 法兰标准 (如: F05/φ50/4-M6)
  - flange_D            # 法兰直径
  - flange_A            # 法兰A尺寸
  - flange_C            # 法兰C尺寸
  - flange_thread       # 法兰螺纹 (如: 4-M6)
  - pneumatic_size      # 气源接口 (如: G1/4")
```

### 🎯 AT系列独有字段
- `manual_override_model` / `manual_override_price` - 手操器配置
- `spare_parts_model` / `spare_parts_price` - 备件配置
- `flange_standard` - 完整法兰标准描述
- `flange_D, flange_A, flange_C` - 详细法兰尺寸
- `flange_thread` - 法兰螺纹规格
- `pneumatic_size` - 气源接口规格

### 💡 AT特殊说明
- **价格策略**：三档价格（常温/低温/高温）
- **附件系统**：支持手操器和备件包配置
- **法兰系统**：完整的法兰尺寸数据
- **弹簧范围**：K8, K10 等标准弹簧配置
- **型号格式**：AT-{作用方式}{尺寸}{弹簧} (如: AT-SR52K8)

```yaml
型号识别:

  - body_size           # 本体尺寸 (GY-052, GY-063等)

法兰连接:
  - flange_standard     # 法兰标准 (如: F05/φ50/4-M6)
  - flange_D            # 法兰直径
  - flange_A            # 法兰A尺寸
  - flange_C            # 法兰C尺寸
  - flange_thread       # 法兰螺纹
  - pneumatic_size      # 气源接口
```

### 🎯 GY系列特点
- **最精简**：仅13个字段，结构最简单
- **单一价格**：仅常温价格，无低温/高温差异
- **无附件**：不涉及手操器、备件等附件
- **标准法兰**：保留基本法兰连接参数
- **作用方式**：主要为SR（单作用）

### 💡 GY特殊说明
- **价格策略**：仅单一常温价格
- **型号格式**：GY-{尺寸}{作用方式} (如: GY-52SR)
- **定位**：经济型产品线，配置精简
- **行程**：标准90°行程

### GY系列
- ✅ 仅需常温价格，结构最简单
