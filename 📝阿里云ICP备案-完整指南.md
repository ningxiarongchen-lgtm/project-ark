## 📋 备案前准备

### 1. 确认域名信息
```
域名：project-ark.com.cn
注册商：阿里云
域名所有者：必须与备案主体一致
```

### 2. 准备材料

#### 个人备案需要：
- ✅ 身份证正反面照片（清晰）
- ✅ 手机号（接收验证码）
- ✅ 邮箱
- ✅ 备案幕布照片（阿里云会提供或用APP拍摄）

#### 企业备案需要：
- ✅ 营业执照副本（彩色扫描件）
- ✅ 法人身份证正反面
- ✅ 网站负责人身份证正反面
- ✅ 网站负责人手机号
- ✅ 企业公章（部分材料需要盖章）
- ✅ 备案幕布照片

### 3. 备案服务号
- 阿里云服务器才能申请备案
- 如果没有服务器，需要购买一个（最便宜的即可）
- **建议：** 购买轻量应用服务器（￥60/年起）

## 🚀 备案详细步骤

### 第1步：登录阿里云备案系统（5分钟）

1. **访问阿里云备案系统**
   ```
   https://beian.aliyun.com
   ```

2. **点击"开始备案"**

3. **选择备案类型**
   - 首次备案：如果从未备案过
   - 新增网站：如果之前备案过其他网站

---

### 第2步：填写主体信息（10分钟）

#### 个人备案填写：
```
主体类型：个人
证件类型：中国大陆居民身份证
证件号码：您的身份证号
主体负责人姓名：您的真实姓名
主体负责人手机号：您的手机号
主体负责人邮箱：您的邮箱
主体负责人证件住所：身份证地址
通信地址：您的实际通信地址
```

#### 企业备案填写：
```
主体类型：企业
证件类型：工商营业执照
证件号码：统一社会信用代码
单位名称：营业执照上的名称
法人姓名：法人真实姓名
法人证件号码：法人身份证号
主体负责人信息：（通常是法人）
通信地址：营业执照地址
```

---

### 第3步：填写网站信息（10分钟）

```
网站名称：智能制造综合管理系统
（或其他符合规范的名称，不能带"中国"等词）

网站域名：project-ark.com.cn

云服务：选择"阿里云"
产品类型：选择您购买的服务类型
备案服务号：填写您的备案服务号

网站内容：根据实际情况选择
- 企业网站
- 企业应用系统

网站负责人信息：
姓名：网站负责人真实姓名
手机号：网站负责人手机号
邮箱：网站负责人邮箱
证件号码：网站负责人身份证号
```

---

### 第4步：上传证件资料（10分钟）

#### 需要上传的照片：

**个人备案：**
1. 身份证正面照片
2. 身份证反面照片
3. 备案核验单（系统生成，签字拍照上传）

**企业备案：**
1. 营业执照副本
2. 法人身份证正反面
3. 网站负责人身份证正反面
4. 备案核验单（系统生成，签字+盖章拍照上传）

**照片要求：**
- 格式：JPG、JPEG、PNG
- 大小：不超过4MB
- 清晰可见，四角完整
- 无反光、无遮挡

---

### 第5步：人脸核验/幕布拍照（5分钟）

阿里云现在支持2种方式：

**方式A：阿里云APP人脸核验（推荐）**
```
1. 下载"阿里云"APP
2. 打开APP，进入备案
3. 选择人脸核验
4. 按提示进行人脸识别
5. 完成！
```

**方式B：幕布拍照**
```
1. 申请阿里云幕布（免费邮寄）
2. 收到后拍照上传
3. 照片要求：背景纯色，人物清晰
```

**推荐：** 用方式A，更快更方便

---

### 第6步：提交备案（1分钟）

1. 检查所有信息
2. 勾选"我已阅读并同意《信息安全管理协议》"
3. 点击"提交备案"

---

### 第7步：阿里云初审（1-2个工作日）

**阿里云会审核您的材料**

可能的结果：
- ✅ 初审通过 → 进入管局审核
- ❌ 初审退回 → 按提示修改后重新提交

**注意：** 保持手机畅通，阿里云可能会电话核实

---

### 第8步：管局审核（10-20个工作日）

**阿里云初审通过后，会提交到省通信管理局**

审核时间：
- 最快：7个工作日
- 通常：10-15个工作日
- 最慢：20个工作日

**期间注意：**
- 保持手机畅通（管局可能电话核实）
- 检查邮箱（会收到审核通知）
- 网站不要访问（备案期间网站不能访问）

---

### 第9步：备案成功（收到通知后）

**备案成功后，您会收到：**
- 短信通知
- 邮件通知（含备案号）

**备案号格式：**
```
例如：京ICP备12345678号-1
```

**收到后立即操作：**
1. 在网站底部添加备案号
2. 备案号链接到：beian.miit.gov.cn
3. 去七牛云修改域名配置：海外 → 中国境内

## ⚠️ 重要注意事项

### 1. 备案期间网站不能访问
```
管局审核期间，网站必须关闭
否则会导致备案失败
```

**解决方案：**
- 先用七牛云海外节点（能访问，但稍慢）
- 备案成功后再切换到国内节点

### 2. 网站名称规范

**个人备案：**
- ❌ 不能带：企业、工作室、网站、平台等
- ❌ 不能带：中国、中华等
- ✅ 可以用：个人博客、某某的网站
- ✅ 建议：XX管理系统、XX应用

**企业备案：**
- ✅ 可以用公司名称
- ✅ 可以用产品名称
- ✅ 建议：智能制造管理系统

### 3. 主体信息必须真实
```
所有信息必须真实有效
假信息会导致备案失败或被注销
```

### 4. 手机号必须畅通
```
阿里云和管局都可能电话核实
务必保持手机畅通
```

### 5. 一个域名只能备案一次
```
如果之前备案过，需要先注销
再重新备案
```

## 📱 备案期间临时方案

**问题：** 备案期间网站不能访问，怎么办？

**方案：** 使用两个域名

```
域名1：project-ark.com.cn（备案中，暂不使用）
域名2：临时域名（七牛云测试域名或另一个域名）

步骤：
1. 先用域名2（海外节点）让用户访问
2. 同时备案域名1
3. 备案成功后，切换到域名1（国内节点）
```

## 🎯 备案成功后的操作

### 1. 在七牛云修改域名配置

```
1. 登录七牛云控制台
2. CDN → 域名管理
3. 找到 project-ark.com.cn
4. 点击"修改配置"
5. 覆盖范围：海外 → 中国境内
6. 保存
```

### 2. 在网站底部添加备案号

```html
<footer>
  <a href="https://beian.miit.gov.cn" target="_blank">
    京ICP备12345678号-1
  </a>
</footer>
```

### 3. 测试访问速度

```

移动端：应该很快 ✅
```
