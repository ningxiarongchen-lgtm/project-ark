## 第2步：添加监控器（3分钟）

### 2.1 登录后的操作
登录后，您会看到Dashboard（仪表盘）

### 2.2 点击"+ Add New Monitor"
位置：页面左上角的绿色按钮

### 2.3 填写监控器配置

**完整配置信息：**

```
┌─────────────────────────────────────────────────────────────┐
│ Monitor Type: HTTP(s)                                       │
│ （下拉选择第一个选项）                                       │
├─────────────────────────────────────────────────────────────┤
│ Friendly Name: Project Ark Backend                         │
│ （给监控器起个名字，方便识别）                              │
├─────────────────────────────────────────────────────────────┤
│ URL (or IP):                                                │
│ https://project-ark-efy7.onrender.com/api/health           │
│ （这是您的后端健康检查地址）                                │
├─────────────────────────────────────────────────────────────┤
│ Monitoring Interval: 5 minutes                              │
│ （免费版最短间隔，足够防止休眠）                            │
├─────────────────────────────────────────────────────────────┤
│ Monitor Timeout (Optional): 30                              │
│ （30秒超时时间）                                            │
├─────────────────────────────────────────────────────────────┤
│ Alert Contacts:                                             │
│ ☑ (选择您注册的邮箱)                                        │
│ （如果后端宕机会发邮件通知您）                              │
└─────────────────────────────────────────────────────────────┘
```

### 2.4 保存配置
点击页面底部的 **"Create Monitor"** 绿色按钮

---

## 第3步：验证配置成功（1分钟）

### 3.1 等待第一次检查
- UptimeRobot会立即发送第一次ping
- 等待约30秒-1分钟

### 3.2 查看状态
在Dashboard页面，您会看到：

```
✅ 成功标志：
┌──────────────────────────────────────┐
│ Project Ark Backend                  │
│ Status: Up (绿色)                    │
│ Uptime: 100%                         │
│ Last Checked: Just now              │
└──────────────────────────────────────┘
```

```
❌ 如果看到红色"Down"：
- 等待5分钟
- 手动刷新页面
- 如果还是Down，检查URL是否正确
```
