# 批量选型Excel模板说明

## 📊 Excel表格列定义

### 必需列（至少需要这些信息）：

| 列名（中文） | 列名（英文） | 说明 | 示例 |
|------------|------------|------|------|
| TAG/位号 | TAG | 设备位号 | 02708-XV-10201 |
| 阀门类型 | Valve Type | 球阀/蝶阀/闸阀 | 球阀, Ball Valve, B822 |
| 阀门尺寸 | Size | DN尺寸 | DN50, DN100, DN150 |
| 扭矩 | Torque | 最大扭矩值(Nm) | 190, 972 |
| 数量 | Quantity | 需要数量 | 1, 2 |

### 可选列：

| 列名 | 说明 | 示例 |
|------|------|------|
| Service | 工艺介质 | 气动液端滑水闸阀 |
| Pressure Rating | 压力等级 | CL150, CL300 |
| Material | 阀体材质 | ASTM A216 WCB |
| 作用方式 | 单作用/双作用 | 双作用, DA |

## 🔧 自动选型逻辑

1. **扭矩计算**：输入扭矩 × 1.5（安全系数）
2. **阀门类型识别**：
   - 球阀：Ball Valve, 球阀, B8xx型号
   - 蝶阀：Butterfly Valve, 蝶阀
   - 闸阀：Gate Valve, 闸阀
3. **尺寸匹配**：从Size列提取DN值
4. **执行器推荐**：
   - 扭矩需求 ≤ 执行器扭矩输出
   - 阀门尺寸 ≤ 执行器适配尺寸

## 📝 支持的表格格式

### 格式1：完整的技术表格（如图片所示）
```
TAG | Service | Model Code | Size | Torque
02708-XV-10201 | 气动液端滑水闸阀 | B822 | DN50 | 190
```

### 格式2：简化表格
```
位号 | 阀门类型 | 尺寸 | 扭矩 | 数量
FV-001 | 球阀 | DN100 | 500 | 1
FV-002 | 蝶阀 | DN150 | 800 | 2
```

### 格式3：最简表格（只有关键信息）
```
扭矩 | 阀门尺寸
190 | DN50
972 | DN150
```

## 🎯 扭矩安全系数说明

- **标准安全系数**：1.5倍
- **计算示例**：
  - 阀门扭矩：190 Nm
  - 选型扭矩：190 × 1.5 = 285 Nm
  - 推荐执行器：≥285 Nm
