#### 1.2 DA（双作用）选型测试 ✅
**测试用例1：球阀 + DA**
```
输入数据：
- TAG: FV-001
- 阀门类型: 球阀
- 阀门尺寸: DN100
- 扭矩: 500
- 安全系数: 1.3
- 作用类型: DA
- 机构类型: Scotch Yoke

验收标准：
- [ ] 选型成功
- [ ] 推荐型号格式：SF-DAxx（不带/C）
- [ ] 扭矩满足：执行器扭矩 >= 500 × 1.3 = 650 Nm
- [ ] 显示对称轭架 (Symmetric)
```

**测试用例2：蝶阀 + DA**
```
输入数据：
- TAG: FV-002
- 阀门类型: 蝶阀
- 扭矩: 800
- 作用类型: DA
- 机构类型: Scotch Yoke

验收标准：
- [ ] 选型成功
- [ ] 推荐型号格式：SF-DAxx/C（带/C）
- [ ] 扭矩满足要求
- [ ] 显示倾斜轭架 (Canted)
```

#### 1.3 SR（单作用）选型测试 ✅

**测试用例3：故障关 (Fail Close)**
```
输入数据：
- TAG: FV-003
- 阀门类型: 球阀
- 开启扭矩: 400
- 关闭扭矩: 450
- 作用类型: SR
- 故障安全位置: Fail Close
- 机构类型: Scotch Yoke

验收标准：
- [ ] 选型成功
- [ ] 推荐型号格式：SF-SRxxxx-STC
- [ ] 满足条件：AST >= 400 且 SET >= 450
- [ ] 显示实际扭矩 = min(AST, SET)
```

**测试用例4：故障开 (Fail Open)**
```
输入数据：
- TAG: FV-004
- 阀门类型: 蝶阀
- 开启扭矩: 350
- 关闭扭矩: 380
- 作用类型: SR
- 故障安全位置: Fail Open
- 机构类型: Scotch Yoke

验收标准：
- [ ] 选型成功
- [ ] 推荐型号格式：SF-SRxxxx/C-STO
- [ ] 满足条件：SST >= 350 且 AET >= 380
- [ ] 显示实际扭矩 = min(SST, AET)
```
