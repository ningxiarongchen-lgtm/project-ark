## 📝 字段说明

### ✅ 必填字段

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `model_base` | 执行器型号 | SF10-150SR3-6 |
| `series` | 系列（SF/AT/GY） | SF |
| `action_type` | 作用类型 | SR（弹簧复位）或 DA（双作用） |

### 📐 尺寸字段（对应图片中的尺寸表）

| 字段名 | 说明 | 单位 | 对应图片列 |
|--------|------|------|-----------|
| `L1` | 单作用总长 | mm | L1 |
| `L2` | 气缸长度 | mm | L2 |
| `m1` | 尺寸参数1 | mm | m1 |
| `m2` | 尺寸参数2 | mm | m2 |
| `A` | 方口尺寸 | mm | A |
| `H1` | 高度1 | mm | H1 |
| `H2` | 高度2 | mm | H2 |
| `D` | 直径尺寸 | mm | D |
| `G` | 气动连接 | - | G（如NPT1/4"） |

### 🔧 其他重要字段

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `connect_flange` | 连接法兰标准 | ISO 5211 F10 |
| `body_size` | 本体尺寸代号 | 10, 12, 14, 16... |
| `cylinder_size` | 气缸尺寸 | 150, 170, 200... |
| `base_price_normal` | 常温价格（元） | 2800 |
| `base_price_low` | 低温价格（元） | 3200 |
| `base_price_high` | 高温价格（元） | 3500 |
| `description` | 描述 | SF10弹簧复位执行器 |

## 🚀 导入步骤

### 步骤1：准备CSV文件

1. **使用提供的模板**
   - 打开 `C-MAX执行器导入模板.csv`
   - 已包含30行示例数据

2. **或创建新文件**
   - 复制模板的表头（第一行）
   - 添加您的数据

3. **数据整理提示**
   - 所有尺寸单位：毫米（mm）
   - 价格单位：元（CNY）
   - 型号必须唯一
   - 保持CSV格式（UTF-8编码）

### 步骤2：登录系统

```
地址: http://localhost:5173 (本地) 或 https://smart-system.pages.dev (线上)
账号: 管理员或技术工程师账号
```

### 步骤3：进入数据管理

```
侧边栏 → "数据管理" → "执行器管理" 标签
```

### 步骤4：批量导入

```
1. 点击 "下载模板" （可选，查看标准格式）
2. 点击 "选择文件（可多选）"
3. 选择一个或多个CSV文件（按住Ctrl/⌘多选）
4. 点击 "开始导入 (N个文件)"
5. 等待导入完成，查看结果
```

## 💡 导入技巧

### 1. 分系列导入
可以将不同系列的数据分成多个文件：
- `SF10系列.csv`
- `SF12系列.csv`
- `SF14系列.csv`
...然后一次性选择所有文件导入

### 2. 更新现有数据
- 勾选 "更新重复数据" 选项
- 系统会根据 `model_base` 字段识别重复项
- 重复项会被更新而不是跳过

### 3. 价格处理
- 如果某个温度等级没有价格，可以留空
- 至少提供一个价格（常温/低温/高温）
- 价格必须为正数

### 4. 尺寸数据
- 所有尺寸都是可选的
- 但建议尽可能填写完整
- 用于选型计算和图纸生成

## ⚠️ 常见问题

### Q1: 导入失败，提示"型号已存在"
**解决**: 勾选"更新重复数据"选项，或修改型号使其唯一

### Q2: 某些字段为空可以吗？
**解决**: 除了必填字段（model_base, series, action_type），其他字段都可以为空

### Q3: 价格字段应该填什么？
**解决**: 
- `base_price_normal`: 常温(-20~80°C)标准价
- `base_price_low`: 低温(-40~80°C或更低)价格
- `base_price_high`: 高温(-20~120°C)价格

### Q4: 连接法兰怎么填？
**解决**: 按图片中的格式填写，如：
- ISO 5211 F10
- ISO 5211 F12
- ISO 5211 F14
...

### Q5: 可以同时导入多个文件吗？
**解决**: 可以！系统支持多文件批量导入，按住Ctrl/⌘选择多个文件

## 📋 快速参考

### 系列代号
- **SF**: 拨叉式气动执行器
- **AT**: 齿轮齿条式（标准）
- **GY**: 齿轮齿条式（重载）

### 作用类型
- **SR**: Spring Return（弹簧复位）
- **DA**: Double Acting（双作用）

### 本体尺寸（SF系列）
- 10, 12, 14, 16, 25, 30, 35, 40, 48, 60

### 连接法兰标准
- ISO 5211 F10/F12/F14/F16/F25/F30/F35/F40/F48/F60

4. 是否有特殊字符导致解析失败
